# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2025-03-06

### Added
- Initial release
- Auto-generated CRUD for MikroORM entities
- Email/password authentication with JWT cookies
- OAuth — Google, GitHub, Microsoft
- Role-based access control (RBAC) with auto-seeded Admin and Viewer roles
- Email verification and MFA support
- Collapsible sidebar with Security section
- Light / dark / system theme toggle
- Fastify plugin API (`createAdminPlugin`)
- CLI commands: `create-admin`, `reset-password`
- Database migrations via MikroORM
